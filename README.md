## 🧰 Techs & Dependencies

### Core Stack

* **Java 17 & Spring Boot 4:** Main Ecosystem.
* **Spring MVC:** REST architecture for scheduling endpoints.
* **Spring Data JPA.** 
* **PostgreSQL:** Relational Database.
* **Flyway:** Migrations and versioning management.

### Security & Performance

* **Spring Security**
* **JWT**
* **Redis & Spring Cache.** 
* **Bean Validation.**

### Documentation & Tests
* **SpringDoc OpenAPI (Swagger).**
* **JUnit 5 & Mockito**
* **H2 Database**
